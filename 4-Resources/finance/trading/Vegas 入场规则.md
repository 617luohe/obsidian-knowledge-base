# Vegas 入场规则

> 基于 Vegas 通道和均线系统的入场信号判断规则。

## 入场的前提条件

任何入场必须同时满足：

1. ✅ **大周期方向确认** — 日线（或操作周期的大一级）144 EMA 方向明确
2. ✅ **价格处于通道合理位置** — 不是在远离通道的地方追入
3. ✅ **12/24 触发线给出信号** — 与趋势方向一致的金叉/死叉
4. ✅ **多周期无冲突** — 至少 2/3 周期方向一致

## 标准入场信号

### A 类信号：通道回踩入场（最可靠）

```
条件：大周期通道方向明确
信号：价格回调至 Vegas 通道区域（144-169之间）
确认：12/24 在通道区域内拐头，与通道同向
入场：在 12/24 确认拐头时入场
止损：通道外侧（169 EMA 外侧）
```

A 类信号的可靠性在于"顺大势、逆小势"——在大趋势方向不变的前提下，利用小周期回调到通道支撑/阻力区域时入场。

### B 类信号：通道突破入场

```
条件：价格从通道内突破 144 EMA（或从外侧突破通道）
信号：价格放量突破通道边界
确认：突破后 12/24 发散跟进
入场：突破确认后入场
止损：通道内侧或最近波段低点/高点
```

### C 类信号：144 首次回踩入场（强趋势中）

```
条件：价格从下方刚上穿 144 EMA，或从上方刚下穿
信号：首次回踩 144 EMA 不破
确认：144 EMA 持续同向倾斜
入场：回踩确认时入场
止损：144 EMA 内侧
```

## 无效信号（不应入场的情况）

| 情况 | 原因 |
|------|------|
| 144 EMA 走平时出现信号 | 趋势不明，假信号高发 |
| 价格远离通道追入 | 乖离过大，回调风险高 |
| 12/24 信号与通道方向相反 | 逆势操作，胜率低 |
| 大周期通道方向不明时 | 无趋势依据 |
| 数据发布前后 | 市场可能剧烈波动 |

## 入场后的持仓逻辑

| 阶段 | 依据 | 操作 |
|------|------|------|
| 初期 | 价格在通道内运行 | 持有，观察 12/24 是否持续顺向 |
| 中期 | 价格沿通道向上/向下运行 | 移动止损至成本位 |
| 末期 | 12/24 反向交叉，或价格突破通道 | 准备出场 |
