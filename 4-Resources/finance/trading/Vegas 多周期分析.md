# Vegas 多周期分析

> 分析 Vegas 通道在不同时间周期（1H/4H/1D/1W）上的表现，利用多周期共振提高交易胜率。这是 Vegas 系统的核心进阶能力。

## 为什么要关注周期级别

Vegas 通道在单一周期上的信号可靠性有限。同样的通道形态，在大周期和小周期上的含义完全不同：

- **大周期**（日线/周线）通道决定**战略方向** — 不可逆
- **小周期**（1H/4H）通道决定**战术时机** — 灵活执行
- **周期冲突**时，以大周期为准，小周期等待

## 标准周期框架

| 周期级别 | 144 EMA 覆盖时长 | 用途 |
|----------|-----------------|------|
| **周线 (1W)** | ~7个月 | 超级趋势方向，机构级别 |
| **日线 (1D)** | ~7个月（交易日） | 主要趋势方向，战略级判断 |
| **4小时 (4H)** | ~24天 | 中期趋势，波段操作 |
| **1小时 (1H)** | ~6天 | 短线交易，精确入场 |
| **15分钟 (15m)** | ~1.5天 | 超短线，一般不用于Vegas系统 |

> 144 EMA 在日线上约覆盖 7 个月交易数据，在 4H 上约覆盖 24 天。周期越小，通道对近期价格的敏感度越高，假信号也越多。

## 多周期共振规则

### 规则一：大周期定方向

以大一级周期的 Vegas 通道方向作为操作方向：

```
日线通道向上 → 只做多
日线通道向下 → 只做空
日线通道横向 → 观望，或仅在小周期短线操作
```

### 规则二：小周期找时机

在顺从大周期的前提下，用较小周期的通道寻找入场点：

```
例：日线通道向上（看多）
  → 切到 4H 等价格回调至 4H Vegas 通道区域
  → 结合 12/24 触发线向上拐头入场
  → 这样"顺大势、逆小势"入场
```

### 规则三：周期冲突时以哪方为准

| 大周期方向 | 小周期信号 | 处理方式 |
|------------|------------|----------|
| ↑ 多头 | ↑ 多头（共振） | ✅ 最强信号，积极操作 |
| ↑ 多头 | ↓ 空头（冲突） | ⏳ 等待小周期回调结束，不追空 |
| ↓ 空头 | ↓ 空头（共振） | ✅ 最强信号，积极操作 |
| ↓ 空头 | ↑ 多头（冲突） | ⏳ 等待小周期反弹结束，不追多 |
| → 横盘 | 任意 | ⚠️ 仅做短线，轻仓 |

### 规则四：通道方向一致性评分

同时检查 3 个周期的通道方向：

| 一致周期数 | 评分 | 操作建议 |
|------------|------|----------|
| 3/3 一致 | ★★★ | 重仓操作，高置信度 |
| 2/3 一致 | ★★ | 正常操作，设好止损 |
| 1/3 一致 | ★ | 轻仓或观望 |
| 0/3 一致 | ☆ | 不操作 |

## 周期切换的具体步骤

```
1. 打开日线 → 判断 144/169 通道方向 → 确定多空方向
2. 打开 4H → 等待价格回调至通道区域
3. 打开 1H → 观察 12/24 触发线，找精确入场点
4. 确认多周期方向一致后入场
```

## 通道在不同周期上的特征

| 特征 | 大周期（日线/周线） | 小周期（1H/4H） |
|------|-------------------|-----------------|
| 通道稳定性 | 高，不易改变方向 | 低，容易转向 |
| 支撑/阻力强度 | 强，价格尊重度高 | 弱，价格容易穿越 |
| 通道宽度 | 宽（点数/金额大） | 窄（点数/金额小） |
| 适用策略 | 趋势跟踪，中长线 | 波段，短线 |
| 假信号率 | 低 | 中高 |

## 实战示例

```
品种：BTC/USD
日线：144 EMA 向上（θ ≈ 30°），价格在通道上方 → 多头方向
4H：价格回踩 144 EMA，触碰到 Vegas 通道上沿
1H：12 EMA 在通道内向上拐头，上穿 24 EMA

解读：三级周期做多共振
→ 入场：1H 12/24 金叉确认时入场做多
→ 止损：4H 通道下沿（169 EMA 下方）
→ 目标：日线通道上方（ until 价格远离日线144 EMA）
```
