# DE 安全剃须刀架横评

> 双面安全剃须刀（DE）的刀架是整个剃须系统的核心。刀头类型、材质、重量和手柄长度直接决定了剃须手感和效果。

## 刀架结构

```
          ┌──── 刀头（Head）────┐
          │                     │
     ┌────┴────┐          ┌────┴────┐
     │  上盖    │          │  底盖   │
     │ (Cap)    │          │ (Base)  │
     └────┬────┘          └────┬────┘
          │                    │
          │    刀片夹在中间     │
          └────────┬───────────┘
                   │
              ┌────┴────┐
              │  手柄    │
              │ (Handle) │
              └─────────┘
```

## 一、刀头类型（最重要选择）

### 1.1 封闭式（Closed Comb / Safety Bar）

刀头底部是一根完整的弧形安全条，刀片露出较少。

| 特性 | 评价 |
|------|------|
| 激进程度 | ⭐⭐（温和） |
| 新手友好 | ⭐⭐⭐⭐⭐ |
| 顺刮体验 | 优秀 |
| 逆刮体验 | 良好 |
| 浓密胡须 | 一般（可能堵塞） |

**推荐给**：**所有人，尤其是新手**。绝大多数人一把封闭式就足够。

代表产品：
- Merkur 34C（HD）— **最经典的新手刀**
- Edwin Jagger DE89 — 与 34C 齐名
- Muhle R89 — 与 DE89 同头，欧洲经典

---

### 1.2 开放式（Open Comb）

刀头底部是梳齿状结构，刀片直接接触皮肤。

| 特性 | 评价 |
|------|------|
| 激进程度 | ⭐⭐⭐（中等） |
| 新手友好 | ⭐⭐ |
| 顺刮体验 | 优秀（尤其粗硬胡须） |
| 逆刮体验 | 一般（较刺激） |
| 浓密胡须 | ⭐⭐⭐⭐⭐ 不堵塞 |

**推荐给**：胡须粗硬、有蓄须习惯（多日一剃）的用户。

**原理**：梳齿让长胡须穿过，不会堵塞刀头。但刀片直接接触皮肤，技术要求更高。

代表产品：
- Merkur 15C / 11C — 经典开放式入门
- Fatip Piccolo — 意大利制造，性价比高
- Muhle R41（激进型） — 最激进的量产 DE 刀头之一

---

### 1.3 斜拉式（Slant Bar / 扭拉式）

刀头呈扭转型，使刀片以斜角切割胡须（类似[[吉列 Slide 剃须法]]的机械实现）。

| 特性 | 评价 |
|------|------|
| 激进程度 | ⭐⭐⭐⭐ |
| 新手友好 | ⭐ |
| 切割效率 | ⭐⭐⭐⭐⭐ 最高 |
| 顺刮体验 | 非常优秀（滑切效应） |
| 浓密胡须 | ⭐⭐⭐⭐⭐ |

**原理**：刀片被扭转后产生倾斜角度，切割时产生"滑切"效应，阻力大幅降低。

**推荐给**：有经验的 DE 用户，胡须粗硬难剃者。

代表产品：
- Merkur 37C / 39C — 最经典的 Slant
- Ikon X3 — 现代 Slant 设计
- Razorock German 37 — 性价比较高的 Slant

---

### 1.4 可调式（Adjustable）

可通过旋钮调整刀片露出量，一把刀架实现从温和到激进的多档调节。

| 特性 | 评价 |
|------|------|
| 调节范围 | 1-9 档（不同品牌范围不同） |
| 新手友好 | ⭐⭐⭐⭐（可从最低档开始） |
| 灵活性 | ⭐⭐⭐⭐⭐ |
| 复杂度 | 结构复杂，相对不耐摔 |

**推荐给**：想"一刀到底"的用户，或不确定自己偏好激进还是温和的新手。

代表产品：

| 品牌 | 型号 | 调节方式 | 价格 | 特点 |
|------|------|----------|------|------|
| Merkur | Progress | 旋钮底端调节 | ¥400-500 | 经典可调，1-5档 |
| Merkur | Futur | 旋钮底端调节 | ¥500-600 | 未来感设计，激进档偏高 |
| Rockwell | 6C / 6S | **换底板**（6档） | ¥300-600 | 通过更换底板达到不同档位 |
| Rex | Ambassador | 旋钮调节 | ¥1200+ | 全不锈钢，高端可调 |

## 二、材质对比

刀架材质直接影响手感、重量、耐用性和价格。

| 材质 | 重量 | 耐用性 | 手感 | 价格 | 代表品牌 |
|------|------|--------|------|------|----------|
| **锌合金压铸（Zamak）** | 中等 | ⭐⭐⭐ | 温润 | ¥80-300 | Merkur, EJ, Muhle |
| **黄铜（Brass）** | 较重 | ⭐⭐⭐⭐⭐ | 扎实 | ¥200-500 | Karve, Fatip |
| **不锈钢（Stainless Steel）** | 重 | ⭐⭐⭐⭐⭐ | 冰冷扎实 | ¥500-1500+ | Rockwell 6S, Rex |
| **钛合金（Titanium）** | 很轻 | ⭐⭐⭐⭐⭐ | 轻巧坚固 | ¥800-2000+ | Timeless, BBS |
| **铝合金（Aluminum）** | 极轻 | ⭐⭐⭐ | 轻 | ¥100-400 | Razorock, Henson |
| **注塑/塑料** | 极轻 | ⭐⭐ | 廉价感 | ¥20-60 | Wilkinson Classic |

### 关于 Zamak

> 绝大多数入门到中端刀架（Merkur、Muhle、Edwin Jagger）使用 **锌合金压铸（Zamak）**。它够用，但有两个缺点：
> 1. 跌落在瓷砖上可能会断裂
> 2. 长期使用后螺纹可能磨损
>
> 如果预算允许且想"一步到位"，不锈钢或黄铜刀架是终身投资。

## 三、手柄选择

### 3.1 长度

| 长度 | 典型值 | 适合 |
|------|--------|------|
| 短柄 | 75-85mm | 面式剃须，操控精准 |
| 标准 | 85-95mm | 大多数人，平衡 |
| 长柄 | 95-110mm | 手大、剃头、喜欢重量感 |

### 3.2 纹路

| 纹路类型 | 防滑性 | 手感 |
|----------|--------|------|
| 光滑 | ⭐ | 湿手易滑 |
| 刻纹（Knurling） | ⭐⭐⭐⭐⭐ | 最佳握持 |
| 菱形纹 | ⭐⭐⭐⭐ | 经典 |
| 凹槽 | ⭐⭐⭐ | 舒适 |

## 六、选购指南

### 按经验推荐

| 经验水平 | 首选推荐 | 价格 | 理由 |
|----------|----------|------|------|
| **新手入门** | **Merkur 34C** 或 **Edwin Jagger DE89** | ¥250 | 封闭式温和、经典易上手 |
| **新手+不想升级** | **Rockwell 6S** | ¥550 | 6档可调，从最温和开始，逐步找到最佳档位 |
| **新手+喜欢轻刀** | **Henson AL13** | ¥450 | 固定角度设计，几乎不会操作失误 |
| **进阶（胡须粗硬）** | **Merkur 37C (Slant)** | ¥300 | 滑切效果，粗硬胡须轻松应对 |
| **进阶（追求效率）** | **Muhle R41** 或 **Fatip Piccolo** | ¥180-280 | 开放式 + 激进，需技术支撑 |
| **高阶（终身投资）** | **Rockwell 6S** 或 **Karve CB** 或 **Timeless** | ¥550-2000 | 全金属材质，可调档位，永不升级 |
| **预算极有限** | **Wilkinson Classic** | ¥30 | ¥30 即可体验 DE 剃须 |

### 按材质推荐

| 需求 | 推荐 |
|------|------|
| **性价比最高** | Zamak（¥200-300）已足够好，Merkur 34C 是标杆 |
| **最耐用** | 不锈钢（Rockwell 6S）或 黄铜（Karve / Fatip） |
| **最轻便/旅行** | 铝合金（Henson AL13 / Razorock） |
| **最轻+最坚固** | 钛合金（Timeless），但价格极高 |

### 常见误区

1. **"越贵越好"**：¥250 的 Merkur 34C 比许多更贵的刀架更适合大多数人
2. **"激进 = 剃得更干净"**：激进度 ≠ 剃净度。正确的角度和手法远比刀架激进度重要
3. **"必须从最温和的开始"**：温和的刀架会让你不自觉地加压，反而养成坏习惯。中等激进的刀架配合零压力手法才是正解
4. **"一把刀架就够了"**：大部分爱好者最终会有 2-3 把轮换，不同刀头类型应对不同场景

### 推荐配置路线

```
入门:     Merkur 34C          ¥250    用 6 个月掌握手法
         └─ 进阶后
              ├ 喜欢温和→ 留 34C 日常
              ├ 想要更高效→ 加一把 Merkur 37C (Slant)   ¥300
              └ 想一步到位→ Rockwell 6S                 ¥550
```
