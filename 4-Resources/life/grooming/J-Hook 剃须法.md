# J-Hook 剃须法

> 针对下巴、喉结等弧度较大部位的进阶剃须技巧，通过 J 型轨迹的勾划动作，在曲面区域实现更好的贴合度。

## 为什么要用 J-Hook

下巴和喉结的弧度使得直上直下的标准手法难以完全贴合：

```
标准手法（直线）：          J-Hook（勾划）：
  ——→                        ⤵
  在曲面上接触不连续       ↙
                             形成连续接触
```

J-Hook 通过改变刀头轨迹，使刀片在曲面上的接触更连续，减少来回调整的次数，从而降低刺激。

## 操作步骤

### 第一步：起始位置

- 将刀头平贴在目标区域（如下巴）的一侧
- 保持标准 30° 角度

### 第二步：J 型轨迹

```
1. 从下往上沿弧线推刀（↗）
2. 到弧顶时，动作自然转向下勾（↘）
3. 整体轨迹呈 J 型
```

- 用**手腕的翻转**完成勾划动作，而非手臂
- 动作要**连贯、流畅**，不要中断
- 长度控制在 3-5cm

### 第三步：刀头旋转

在勾划过程中，刀头会沿弧面自然旋转。这是 J-Hook 的关键——不要主动转动刀头，而是让手腕带动刀头沿弧线走，刀头会自然适应曲面。

## 适用部位

| 部位 | J-Hook 方向 | 说明 |
|------|-------------|------|
| 下巴 | 从下往上勾 | 下巴是最常用的 J-Hook 部位 |
| 喉结 | 从侧向上勾 | 注意喉结两侧的凹陷区域 |
| 下颌线 | 沿下颌线勾划 | 从下颚角向中心方向 |
| 颧骨下方 | 从外向内勾 | 脸颊凹陷处 |

## 注意事项

- **仅限熟练者**：新手先掌握[[直上直下剃须法]]后再尝试
- **仅在第二/三遍使用**：第一遍顺刮时不需要 J-Hook
- **勾划幅度不宜过大**：过大会增加割伤风险
- **刀刃锋利度需足够**：钝刀片在勾划时更容易拉扯
