# 剃须刷横评

> 剃须刷是湿剃体验的核心工具之一，负责起泡、涂抹和软化胡须。不同材质的刷毛在软硬度、保水性、起泡速度和价格上天差地别。

## 刷毛类型一览

```
          ┌────────────────────────────────────────────────┐
          │                   剃须刷类型                    │
          ├────────────────────────────────────────────────┤
          │                                                │
  人造纤维 ──  Synthetic (合成纤维)                         │
          │   ├─ 普通尼龙 → 现代纤维（如 Muhle Silvertip）  │
  动物毛 ────  Boar (猪鬃)                                 │
          │   ├─ Badger (獾毛)                             │
          │   │   ├─ Pure / Best / Super / Silvertip       │
          │   └─ Horse (马毛)                              │
          └────────────────────────────────────────────────┘
```

## 详细分类

### 1. 合成纤维刷（Synthetic）

目前最受新手推荐的类型，技术进步使其性能已接近中端动物毛。

| 特性 | 评价 |
|------|------|
| 软硬度 | 极软（触感柔滑） |
| 保水性 | 较差（含水量低，出水快） |
| 起泡速度 | ⭐⭐⭐⭐⭐ 最快 |
| 耐用性 | 极高，不发臭，不脱落 |
| 价格 | ¥30-200 |
| 代表品牌 | Muhle（Silvertip Fibre）、Yaqi、Razorock、Omega Syntex |
| 干刷适用 | ✅ 可干刷 |

**优点**：
- 无需浸泡，沾水即用
- 无异味，无动物毛腥味
- 价格最低，入门零门槛
- 不易滋生细菌

**缺点**：
- 保水性差，需频繁蘸水
- 泡沫温度保持差
- 部分廉价纤维偏"弹性"而非真实刷感

**选购建议**：预算 ¥50-80 的 Yaqi 或 Razorock 已足够日常使用。

---

### 2. 猪鬃刷（Boar / 野猪毛）

传统湿剃的经典选择，价格便宜但需要"开刷"期。

| 特性 | 评价 |
|------|------|
| 软硬度 | 偏硬，使用后变软 |
| 保水性 | 良好，吸水后膨胀分叉 |
| 起泡速度 | ⭐⭐⭐⭐ |
| 开刷期（Break-in） | 约 2-4 周（新刷有猪鬃味，需浸泡使用） |
| 价格 | ¥30-150 |
| 代表品牌 | Omega、Semogue |

**优点**：
- 性价比极高，起泡快
- 刷毛末端分叉后非常柔软
- 保水好，适合硬皂起泡
- 越用越好用

**缺点**：
- 新刷有猪鬃腥味（使用后逐渐消除）
- 需提前浸泡 5-10 分钟
- 刷毛偏硬，敏感皮肤初期可能不适
- 使用寿命比合成纤维短（1-3 年）

**选购建议**：Omega 10049 (Pro 49) 约 ¥50，是最经典入门的猪鬃刷。

---

### 3. 獾毛刷（Badger）

湿剃界的"高端代表"，根据毛质分多个等级。

**分级体系（由低到高）：**

| 等级 | 特点 | 价格区间 | 说明 |
|------|------|----------|------|
| **Pure**（纯獾毛） | 偏硬，毛质不纯，有倒刺 | ¥80-150 | 入门级獾毛，不如好的猪鬃 |
| **Best**（最佳獾毛） | 软硬适中，刷尖白身黑 | ¥150-350 | **性价比最高的獾毛等级** |
| **Super**（超级獾毛） | 柔软，毛尖长，保水优秀 | ¥300-600 | 舒适度明显提升 |
| **Silvertip**（银尖） | 极软，顶尖纯白，保水顶级 | ¥500-2000+ | 獾毛的顶级形态 |

| 特性 | 评价 |
|------|------|
| 软硬度 | 因等级而异（Pure 硬 → Silvertip 极软） |
| 保水性 | ⭐⭐⭐⭐⭐ 最佳 |
| 起泡速度 | ⭐⭐⭐ |
| 价格 | ¥80-2000+ |
| 代表品牌 | Simpson、Edwin Jagger、Kent、Muhle |

**优点**：
- 保水能力极强，泡沫温度保持好
- Silvertip 级别的触感是最高享受
- 起泡厚实绵密
- 使用寿命最长（5-10年+）

**缺点**：
- 价格昂贵（好的 Silvertip 轻松上千）
- 需浸泡 + 甩干
- 动物毛特有气味（比猪鬃淡）
- 需要护理（通风干燥、定期清洗）

**选购建议**：不建议买 Pure 级（不如买好猪鬃），直接从 Best 起步。Simpson 的 Best Badger 是行业标杆。

---

### 4. 马毛刷（Horse Hair）

小众选择，性能介于猪鬃和獾毛之间。

| 特性 | 评价 |
|------|------|
| 软硬度 | 中等偏软 |
| 保水性 | 中等 |
| 起泡速度 | ⭐⭐⭐ |
| 价格 | ¥50-200 |
| 代表品牌 | Vie-Long |

**优点**：
- 无须开刷，出厂即用
- 比猪鬃软，比獾毛便宜
- 动物毛质感，价格亲民

**缺点**：
- 马毛味留存时间较长
- 毛量偏少，起泡略薄
- 品牌选择较少

**选购建议**：Vie-Long 是马毛刷的绝对主力品牌，适合想尝试动物毛但预算有限的用户。

## 选购指南

### 按预算推荐

| 预算 | 推荐 | 理由 |
|------|------|------|
| **¥30-80** | Omega 猪鬃 或 Yaqi 合成纤维 | 猪鬃耐用，合成纤维方便 |
| **¥80-200** | Yaqi 高端合成 / Omega/Semogue 猪鬃 / Vie-Long 马毛 | 选择最多 |
| **¥200-400** | Simpson Best / Muhle 合成 | 獾毛入门最佳区间 |
| **¥400-800** | Simpson Super / Kent BK 系列 / Muhle Silvertip Fibre | 兼顾享受和耐久 |
| **¥800+** | Simpson Silvertip / Kent Silvertip | 终极奢华体验 |

### 按使用场景

| 场景 | 推荐 |
|------|------|
| **出差/旅行** | 合成纤维（干得快、无需浸泡） |
| **硬皂起泡** | 猪鬃（保水好、起泡快） |
| **敏感皮肤** | Silvertip 獾毛 或 高端合成纤维 |
| **日常面式起泡** | 獾毛 Best 或 优质合成纤维 |
| **预算有限入门** | Omega 猪鬃（¥40-60 体验最佳） |

### 常见误区

1. **"獾毛一定比猪鬃好"**：Pure 级獾毛不如 Omega 的优质猪鬃，等级比材质更重要
2. **"合成纤维是廉价替代品"**：高端合成（Muhle Silvertip Fibre、Yaqi）已逼近 Silvertip 獾毛表现
3. **"贵的就是最好的"**：最贵不一定最适合。合成纤维方便、猪鬃起泡强、獾毛体验好，各有侧重
4. **"刷子不需要护理"**：动物毛刷用后要彻底清洗、挤干、倒置通风
