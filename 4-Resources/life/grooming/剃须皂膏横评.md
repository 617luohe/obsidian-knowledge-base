# 剃须皂/膏横评

> 剃须皂和剃须膏是湿剃的核心消耗品，直接决定泡沫的润滑度、保护和香味。从剂型到配方，选择非常丰富。

## 产品形态分类

```
                  ┌────────────────────────────────────┐
                  │          剃须产品形态               │
                  ├────────────────────────────────────┤
                  │                                    │
  硬皂 ───────────┤  硬皂（Hard Soap / Puck）           │
  软皂 ───────────┤  软皂（Soft Soap / Croap）          │
  膏 ─────────────┤  剃须膏（Shaving Cream）             │
  罐装 ───────────┤  罐装泡沫（Canned Foam / Aerosol）  │
  油 ─────────────┤  剃须油（Shaving Oil）               │
                  └────────────────────────────────────┘
```

## 一、剃须皂（Shaving Soap）

### 1.1 硬皂（Hard Soap / Puck）

传统圆饼状，最耐用的形态。

| 特性 | 评价 |
|------|------|
| 形态 | 固体圆饼，装在皂碗中或单独出售 |
| 起泡难度 | ⭐⭐⭐ 需较多时间和技巧 |
| 润滑度 | ⭐⭐⭐⭐⭐ |
| 耐用性 | 极耐用，一块可用 3-6 个月 |
| 价格 | ¥50-150 / 块 |
| 推荐搭配 | 猪鬃刷（起泡效率高）或 獾毛刷（保水好） |

**优点**：
- 最耐用、最经济
- 起泡可控制性最强，能打出非常厚实的泡沫
- 通常含较多护肤成分

**缺点**：
- 起泡最慢，需要耐心和技巧
- 需要起泡碗（干蹭效率低）
- 对水质敏感（硬水起泡困难）

**代表产品**：

| 品牌 | 国家 | 特点 | 价格 | 润滑度 |
|------|------|------|------|--------|
| **Mitchell's Wool Fat** | 英国 | 含羊脂，极润滑 | ¥80 | ⭐⭐⭐⭐⭐ |
| **Tabac** | 德国 | 经典皂味，性能均衡 | ¥80 | ⭐⭐⭐⭐ |
| **Stirling** | 美国 | 极多香味选择 | ¥60 | ⭐⭐⭐⭐⭐ |
| **Haslinger** | 奥地利 | 成分简单，温和 | ¥60 | ⭐⭐⭐⭐ |
| **Martin de Candre** | 法国 | 顶级手工皂 | ¥300+ | ⭐⭐⭐⭐⭐ |

---

### 1.2 软皂（Soft Soap / Croap / 半皂）

质地介于硬皂和膏之间，像软奶酪或面霜。

| 特性 | 评价 |
|------|------|
| 起泡难度 | ⭐⭐ 容易 |
| 润滑度 | ⭐⭐⭐⭐ |
| 耐用性 | 中等，一罐约 2-3 个月 |
| 价格 | ¥60-200 / 罐 |
| 推荐搭配 | 任何刷子 |

**优点**：
- 起泡比硬皂快很多
- 容易控制水量
- 兼顾硬皂的润滑和膏的便捷

**缺点**：
- 比硬皂消耗快
- 罐装运输不便

**代表产品**：

| 品牌 | 国家 | 特点 | 价格 |
|------|------|------|------|
| **Barrister & Mann** | 美国 | 顶级配方，Excelsior 基座 | ¥150 |
| **Cella** | 意大利 | 杏仁味经典，性价比王者 | ¥60 |
| **Proraso (皂)** | 意大利 | 绿/红/蓝三色可选 | ¥50 |
| **Wholly Kaw** | 美国 | 成分顶级，护肤效果出众 | ¥180 |
| **Zingari Man** | 美国 | 专为敏感皮肤设计 | ¥150 |

## 二、剃须膏（Shaving Cream）

### 2.1 管装膏（Tube Cream）

牙膏状挤出使用。

| 特性 | 评价 |
|------|------|
| 起泡难度 | ⭐ 最简单 |
| 润滑度 | ⭐⭐⭐ |
| 耐用性 | 一支约 1-2 个月 |
| 价格 | ¥30-80 / 支 |
| 推荐搭配 | 任何刷子 |

**优点**：
- 起泡最快，挤出黄豆大小即可
- 起泡量大（管装通常非常浓缩）
- 旅行方便

**缺点**：
- 润滑度通常不如优质硬皂
- 含水分较高，泡沫偏轻

**代表产品**：

| 品牌 | 特点 | 价格 |
|------|------|------|
| **Proraso (管装)** | 绿管（薄荷）/ 红管（檀香）/ 白管（敏感） | ¥40-50 |
| **Taylor of Old Bond Street (TOBS)** | 经典英式，多种香味，起泡易 | ¥60-80 |
| **Truefitt & Hill** | 英国皇室御用 | ¥80-120 |
| **Geo F. Trumper** | 经典英式，老牌 | ¥80-120 |
| **Creams (Palmolive / Nivea)** | 超市平价货 | ¥15-25 |

### 2.2 罐装气雾泡沫（Canned Foam / Aerosol）

超市最常见的剃须泡沫，压力罐直接喷出。

| 特性 | 评价 |
|------|------|
| 起泡难度 | 无需，直接喷出 |
| 润滑度 | ⭐⭐ |
| 耐用性 | 一罐约 1-2 个月 |
| 价格 | ¥15-30 |
| 推荐搭配 | 不需要刷子 |

**优点**：
- 最便捷：按压即用
- 无需刷子、无需碗
- 适合出差/应急

**缺点**：
- 润滑度远不如皂和膏
- 泡沫稀薄，保护性差
- 干燥感强，剃须后刺激大

**代表产品**：吉列 Foamy、妮维雅男士、阿迪达斯等。

> **一句话**：罐装泡沫是"能用"但远远"不好用"。如果你在用 DE 剃须，建议至少升级到管装膏。

## 三、配方：动物脂 vs 植物基

剃须皂/膏的核心成分分类。

### 3.1 动物脂皂（Tallow-based）

传统配方，主要脂肪来源为牛脂/羊脂。

| 优点 | 缺点 |
|------|------|
| 润滑度更高，泡沫厚实绵密 | 素食者不适用 |
| 须后保湿效果更好 | 部分人对动物脂敏感 |
| 承受水稀释能力强 | 部分品牌有动物脂味 |

**代表品牌**：Mitchell's Wool Fat（羊脂）、Tabac、Barrister & Mann、Stirling、Wholly Kaw

### 3.2 植物基皂（Vegan / Plant-based）

使用植物油脂，如椰子油、棕榈油、橄榄油、乳木果油等。

| 优点 | 缺点 |
|------|------|
| 纯素友好 | 部分配方润滑度不及动物脂 |
| 成分更现代，护肤成分丰富 | 稳定泡沫的难度稍高 |
| 对敏感皮肤更友好 | |

**代表品牌**：Martin de Candre、Haslinger、Proraso（部分）、Zingari Man、Southern Witchcrafts

### 3.3 对比

| 维度 | 动物脂皂 | 植物基皂 |
|------|----------|----------|
| 泡沫厚度 | ⭐⭐⭐⭐⭐ | ⭐⭐⭐ |
| 润滑度 | ⭐⭐⭐⭐⭐ | ⭐⭐⭐ |
| 护肤成分 | ⭐⭐⭐ | ⭐⭐⭐⭐⭐ |
| 敏感肌友好 | ⭐⭐⭐ | ⭐⭐⭐⭐⭐ |
| 硬水表现 | ⭐⭐⭐⭐ | ⭐⭐⭐ |

## 五、选购指南

### 按预算推荐

| 预算 | 推荐 | 理由 |
|------|------|------|
| **¥15-30** | 管装 Palmolive / 妮维雅 | 入门级但比罐装好得多 |
| **¥40-60** | **🏆 Proraso 管装** 或 **Cella 软皂** | 新手最佳起点，性价比最高 |
| **¥60-100** | **Tabac** 或 **Stirling** | 进阶选择，润滑度大幅提升 |
| **¥100-200** | **Barrister & Mann** 或 **Wholly Kaw** | 顶级配方，极致体验 |
| **¥200+** | **Martin de Candre** | 奢侈手工皂，仪式感 |

### 按使用习惯推荐

| 场景 | 推荐 |
|------|------|
| **快速日常剃须（< 5分钟起泡）** | 管装膏（Proraso / TOBS） |
| **慢节奏享受剃须** | 硬皂（Stirling / Tabac / MWF） |
| **面式起泡** | 软皂（Cella / B&M），直接在面部打圈 |
| **旅行** | 管装膏 或 硬皂切块装小盒 |
| **敏感皮肤** | Zingari Man / Wholly Kaw / Haslinger |
| **追求泡沫颜值** | Barrister & Mann（最上镜的泡沫） |

### 推荐路线

```
新手:
  Proraso 绿管（管装） ¥40  → 学习起泡技巧
         │
         习惯刷子后:
  Cella 红罐（软皂）¥60  → 体验动物脂的润滑
         │
         想进一步升级:
  Stirling（硬皂）¥60  → 大量香味可选 + 顶级润滑
         │
         追求极致:
  Barrister & Mann ¥150 或 Martin de Candre ¥300+
```
