# 吉列 Slide 剃须法

> 吉列（Gillette）命名的斜向滑动剃须技巧，通过在直上直下的过程中加入水平滑动分量，使刀片以更顺滑的角度切割胡须。

## 原理

Slide 剃须的核心在于：在刀头直上直下移动的同时，加入沿刀片刃口方向的滑动分量，使刀片产生"斜切"效果，类似刀切番茄时的滑动切割。

```
标准直上直下：         吉列 Slide：
    ↑                    ↗
    ↑    （仅垂直）      ↗   （垂直+水平分量）
    ↑                    ↗
  刀片垂直切割        刀片斜切 → 切割更顺滑
```

斜切效果降低了每次切割的阻力，使剃须更顺滑，尤其适合胡须粗硬的用户。

## 操作步骤

### 第一步：起始位置

- 刀头贴合皮肤，标准 30° 角度
- 刀头方向对准垂直方向

### 第二步：加入斜向分量

```
推刀方向：  ↑  （垂直向上）
同时滑动：  →  （水平向右/左）

合成方向：  ↗  （斜向上方）
```

具体做法：

1. **向下剃时**：刀头向下移动，同时沿刃口方向滑动（向左或向右）
2. **向上剃时**：刀头向上移动，同时滑动
3. 滑动幅度约 **1-2cm**（在 2-3cm 的 stroke 中）

### 第三步：控制比例

| 垂直：水平比例 | 风格 | 适用场景 |
|---------------|------|----------|
| 3:1 | 保守 Slide | 新手入门，轻度斜切 |
| 2:1 | 标准 Slide | 日常剃须，平衡顺滑与掌控 |
| 1:1 | 激进 Slide | 老手使用，最大顺滑度但难控制 |

## 技巧要点

- Slide 的滑动方向**可以随时切换**，不必固定向左或向右
- 使用 Slide 时刀头角度**稍微放平**（25-30°）效果更佳
- Slide 动作主要来自**手腕的微调**，而非整个手臂的移动
- 如果感觉刀片"卡顿"，增加滑动幅度或换用更锋利的刀片

## 注意事项

| 注意 | 原因 |
|------|------|
| 滑动幅度勿过大 | 过大滑动会使刀头脱离接触，增加割伤风险 |
| 保持刀头稳定 | Slide 中的抖动会直接导致划伤 |
| 不要在逆刮时用激进 Slide | 逆刮 + 大幅 Slide = 极高刺激 |
| 钝刀片慎用 | Slide 对刀片锋利度要求比直上直下高 |
