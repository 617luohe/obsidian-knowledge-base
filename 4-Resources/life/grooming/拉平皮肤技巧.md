# 拉平皮肤技巧

> 剃须前通过手指拉伸使皮肤表面平整，为刀片创造平坦的切割平面，减少割伤和刺激的风险。

## 为什么要拉平皮肤

面部并非完全平坦，在曲面和不平整区域，刀片容易产生跳跃或刮伤：

```
未拉平：                  拉平后：
  ┌─── curved ───┐         ───── flat ─────
  刀片在凸面接触不稳      刀片稳定滑过
  容易跳跃/割伤           切割均匀
```

## 各部位手法

| 部位 | 手法 | 说明 |
|------|------|------|
| **脸颊** | 一手将面部皮肤向后上方提拉 | 用非持刀手，从耳前向后拉 |
| **上唇** | 上唇向下拉，或鼻子向上推 | 用手指压住鼻翼向上提 |
| **下巴** | 下巴皮肤向下拉 | 用非持刀手将下巴皮肤向下拉伸 |
| **喉结** | 头向一侧倾，皮肤向同侧拉 | 喉结处皮肤最紧，拉平效果明显 |
| **下颌线** | 手指沿下颌线向外推 | 从下巴向耳垂方向推开 |

## 单手 vs 双手

### 单手拉平（Free-hand Stretching）

非持刀手负责拉平，持刀手负责剃须：

```
持刀手 → 剃须
另一手 → 食指+拇指撑开皮肤
```

- 适用于大多数部位
- 需要两手协调配合

### 双手拉平（Two-hand Stretching）

某些部位需要双手同时拉平，此时用刀片重量而非持刀手压力：

```
左手 → 从左侧拉平
右手 → 从右侧拉平
刀片 → 利用自重滑过
```

- 适用于额头、喉结等大面积曲面
- 难度较高，需确保刀片角度稳定

## 常见错误

| 错误 | 后果 | 正确做法 |
|------|------|----------|
| 过度拉伸 | 皮肤变形，胡须角度改变 | 轻微拉平即可 |
| 拉伸方向与剃向冲突 | 拉平效果被抵消 | 确保拉伸方向垂直于剃向 |
| 拉平后不释放 | 皮肤持续紧张 | 每 stroke 后放松、重新拉平 |
| 持刀手也参与拉平 | 刀头压力失控 | 只用非持刀手拉平 |
