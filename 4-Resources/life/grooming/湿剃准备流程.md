# 湿剃准备流程

> 湿剃（Wet Shaving）的完整须前准备流程：从清洁、软化到起泡，为刀片创造最佳的滑动条件。

## 完整流程

```
温水洗脸 → 须前油 → 起泡（皂+刷）→ 涂抹 → 剃须
  ①         ②           ③          ④        ⑤
```

## 第一步：温水清洁

- 用 **温热水** 洗脸，打开毛孔、软化胡须
- 可在淋浴后进行，蒸汽已充分软化胡须
- 至少让水接触面部 **2-3 分钟**，确保胡须充分吸水

## 第二步：须前油（Pre-shave Oil）

### 作用

| 作用 | 说明 |
|------|------|
| **润滑** | 在皮肤表面形成油膜，减少刀片摩擦 |
| **软化** | 帮助进一步软化胡须角质 |
| **保护** | 减少刀片直接刮擦带来的刺激 |

### 使用方法

1. 滴 **3-5 滴** 须前油于掌心
2. 双掌搓匀后按在湿润的面部
3. 轻微按摩 **15-20 秒**，覆盖全脸
4. 不要擦干，直接上剃须泡沫

### 配方参考

须前油常见成分：**蓖麻油、橄榄油、葡萄籽油、薄荷精油**，也可用椰子油等替代。

## 第三步：剃须皂/膏与刷

### 剃须刷类型

| 刷毛类型 | 软硬度 | 保水性 | 价格 | 推荐度 |
|----------|--------|--------|------|--------|
| **獾毛（Badger）** | 软 | 优秀 | 高 | 最佳体验 |
| **野猪毛（Boar）** | 偏硬 | 良好 | 中 | 高性价比 |
| **合成纤维（Synthetic）** | 极软 | 一般 | 低 | 新手首选 |
| **马毛（Horse）** | 中等 | 良好 | 中 | 小众选择 |

### 起泡方法

#### 碗式起泡（Bowl Lathering）

```
1. 剃须刷在热水中浸泡 1-2 分钟
2. 甩掉多余水分（保持湿润但不滴水）
3. 刷头在剃须皂表面打圈 10-15 秒，沾取皂质
4. 移至起泡碗中，加水（逐滴）反复搅拌
5. 至泡沫呈"酸奶状"浓稠、有光泽即可
```

#### 面式起泡（Face Lathering）

```
1. 刷子浸泡、甩干同上
2. 直接在面部打圈起泡
3. 边打圈边逐滴加水
4. 泡沫直接在面部形成
```

推荐新手从碗式起泡开始，容易控制水量。

### 理想泡沫标准

- 光泽感（不是干涩的白色）
- 浓稠不滴落
- 手指划过时有明显润滑感
- 不能有大气泡（说明水太多）

## 第四步：涂抹

- 用刷子将泡沫均匀涂抹在整个剃须区域
- 打圈方式让泡沫深入胡须根部
- 每遍剃须前重新涂抹

## 不同场景的选择

| 场景 | 建议 |
|------|------|
| **日常快速剃须** | 温水 + 剃须膏（管装）+ 合成毛刷 |
| **享受式慢剃** | 淋浴 + 须前油 + 獾毛刷 + 剃须皂 |
| **敏感皮肤** | 温水 + 须前油（镇静型）+ 温和剃须皂 |
| **出差/旅行** | 须前油（可替代泡沫）+ 方便剃须 |

## 常见错误

| 错误 | 后果 |
|------|------|
| 水太少 | 泡沫干涩、拉扯感 |
| 水太多 | 泡沫稀薄、无保护 |
| 刷子未泡透 | 刷毛硬、起泡慢 |
| 省掉须前油 | 摩擦增大，后续刺激风险高 |
| 冷水起泡 | 皂不易溶解、泡沫粗糙 |
