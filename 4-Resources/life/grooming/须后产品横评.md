# 须后产品横评

> 须后产品是剃须后对皮肤的修复与保护。从含酒精的须后水到滋润的须后膏，不同类型的须后产品在功效、使用感和适用人群上差异显著。

## 产品类型一览

```
                  ┌──────────────────────────────────────┐
                  │           须后产品类型                 │
                  ├──────────────────────────────────────┤
                  │                                      │
  须后水 ─────────┤  Aftershave Splash / Lotion           │
  须后膏 ─────────┤  Aftershave Balm / Cream              │
  须后凝胶 ───────┤  Aftershave Gel                       │
  明矾块 ─────────┤  Alum Block                           │
  止血笔 ─────────┤  Styptic Pencil                       │
  金缕梅 ─────────┤  Witch Hazel                          │
                  └──────────────────────────────────────┘
```

## 一、须后水（Aftershave Splash）

最传统、最常见的须后产品，多为酒精基底。

| 特性 | 评价 |
|------|------|
| 质地 | 液体，酒精基底 |
| 使用感 | 清爽，涂抹时有明显刺痛 |
| 主要作用 | 消毒杀菌 + 收敛毛孔 |
| 保湿 | ⭐ |
| 镇静 | ⭐⭐ |
| 香味持久 | ⭐⭐⭐⭐ |
| 适合肤质 | 油性皮肤、夏季 |
| 价格 | ¥30-200 |

**优点**：清爽、杀菌、经典男人味。
**缺点**：酒精刺痛，干性皮肤使用后需额外保湿。

**代表品牌**：

| 品牌 | 特点 | 价格 | 说明 |
|------|------|------|------|
| **Proraso 绿（须后水）** | 薄荷+桉树，冰爽感 | ¥50-70 | 最经典入门须后水 |
| **Proraso 红（须后水）** | 檀香，温暖感 | ¥60-80 | 适合秋冬 |
| **Proraso 白（须后水）** | 敏感肌配方，减少酒精刺激 | ¥60-80 | 绿色款刺激性更低 |
| **Floid** | 西班牙经典，浓郁香型 | ¥80-150 | 气味持久，barbershop 味 |
| **Clubman Pinaud** | 美式经典，极浓香 | ¥50-80 | 复古 barbershop 香味 |
| **Fine Accoutrements** | 高品质复刻香型 | ¥80-120 | 质感上乘 |
| **Barrister & Mann** | 顶级配方，含护肤成分 | ¥100-180 | 酒精基但含芦荟等舒缓成分 |

## 二、须后膏（Aftershave Balm）

乳液状，侧重保湿和修复，无酒精或低酒精。

| 特性 | 评价 |
|------|------|
| 质地 | 乳液/膏状 |
| 使用感 | 温和无刺痛，吸收后滋润 |
| 主要作用 | 保湿修复、舒缓刺激 |
| 保湿 | ⭐⭐⭐⭐⭐ |
| 镇静 | ⭐⭐⭐⭐⭐ |
| 香味持久 | ⭐⭐⭐ |
| 适合肤质 | **干性皮肤、敏感皮肤、冬季** |
| 价格 | ¥40-250 |

**优点**：保湿修复效果最好，适合日常使用。
**缺点**：不如须后水清爽，油性皮肤可能觉得粘腻。

**代表品牌**：

| 品牌 | 特点 | 价格 |
|------|------|------|
| **Nivea Sensitive 须后膏** | 🏆 **性价比之王**，敏感肌配方 | ¥25-40 |
| **Proraso 绿（须后膏）** | 薄荷清爽 | ¥50-70 |
| **Proraso 白（须后膏）** | 燕麦配方，舒缓 | ¥50-70 |
| **L'Occitane 须后膏** | 乳木果油，滋养 | ¥150-200 |
| **Truefitt & Hill 须后膏** | 英式，高端体验 | ¥120-200 |
| **Zingari Man 须后膏** | 顶级护肤成分 | ¥150-200 |
| **Stirling 须后膏** | 多种香型可选 | ¥60-90 |

## 三、须后凝胶（Aftershave Gel / Balm Gel）

介于水和膏之间的凝胶质地。

| 特性 | 评价 |
|------|------|
| 质地 | 透明凝胶（常需冷藏使用） |
| 使用感 | 冰爽镇静，吸收快 |
| 主要作用 | 降温镇静、轻度保湿 |
| 适合肤质 | 夏季、剃须刺激明显时 |
| 价格 | ¥40-100 |

代表产品：Proraso 绿须后凝胶、L'Occitane 须后凝胶。

## 四、明矾块（Alum Block）

矿物质晶体，作用更偏向"功能工具"而非"护理产品"。

| 特性 | 评价 |
|------|------|
| 成分 | 硫酸铝钾（天然矿物） |
| 使用感 | 刺痛感，涂抹后停留 30 秒冲洗 |
| 主要作用 | **止血 + 收缩毛孔 + 反馈剃须质量** |
| 保湿 | ❌（反而略有干燥） |
| 价格 | ¥15-40 / 一块用一年以上 |

**优点**：
- 最便宜的须后产品
- 剃须质量反馈工具（刺痛程度反映手法问题）
- 天然成分，无香精酒精
- 一块用非常久

**缺点**：
- 无保湿效果，需叠加其他须后产品
- 刺痛感明显
- 部分人长期使用皮肤干涩

**选购建议**：淘宝 20 元以内的明矾块和 100 元的没有本质区别，都是硫酸铝钾。

## 五、其他类型

### 5.1 止血笔（Styptic Pencil）

专为止血设计的笔状产品，主要成分为硫酸铝。

- 用于割伤时的**局部止血**
- 涂抹时非常刺痛但止血效果极快
- 常备一支在剃须时应急

**价格**：¥10-30

### 5.2 金缕梅水（Witch Hazel）

天然植物提取物，温和收敛。

- 天然酒精（非人工添加），温和
- 兼具须后水的收敛效果和较低刺激性
- 可 DIY：超市金缕梅水即可
- 不少须后水以金缕梅为基底代替酒精

**价格**：¥20-50（超市开架）

### 5.3 芦荟胶（纯）

- 纯芦荟，极致温和
- 镇静修复效果极佳
- 适合皮肤严重刺激时使用
- 超市开架¥20-40

## 七、选购指南

### 标准须后流程

```
冷水冲面 → 轻拍干 → 明矾块（可选）→ 停留30秒 → 冲洗 → 须后水/膏
    ①          ②          ③              ④        ⑤       ⑥
```

### 按肤质推荐

| 肤质 | 推荐产品 | 说明 |
|------|----------|------|
| **油性皮肤** | 须后水（Proraso 绿 / Floid） | 清爽收敛，避免过油 |
| **干性皮肤** | 须后膏（Nivea Sensitive / L'Occitane） | 保湿为主 |
| **敏感皮肤** | 须后膏（Nivea / Zingari Man）或 金缕梅水 | 避免酒精刺激 |
| **正常皮肤/通用** | 须后水夏季 + 须后膏冬季 | 轮换使用 |
| **追求极简** | 明矾块（一块搞定，¥20） | 最经济，但需叠加保湿 |

### 按预算推荐（完整须后方案）

| 预算 | 推荐组合 | 总价 |
|------|----------|------|
| **¥0** | 冷水 → 拍干 | ¥0 |
| **¥20** | 明矾块 | ¥20 |
| **¥50** | Nivea Sensitive 须后膏 | ¥30-40 |
| **¥100** | Proraso 绿须后水 + 明矾块 | ¥70-90 |
| **¥150** | Proraso 白（须后水+须后膏） | ¥120-140 |
| **¥200+** | Barrister & Mann 须后水 + Zingari Man 须后膏 | ¥150-200 |

### 核心建议

> 对大多数 DE 用户来说，**一支 Nivea Sensitive 须后膏（¥30）就已经足够好**。它的成分和效果对标许多 ¥100+ 的"高端"须后膏。如果要升级，建议把钱花在须后水（获得不同香味的体验），而非须后膏。
