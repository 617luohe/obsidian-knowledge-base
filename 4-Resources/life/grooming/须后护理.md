# 须后护理

> 剃须完成后对皮肤的修复和保护程序，包括冷冲洗、止血、须后产品使用等步骤。

## 完整流程

```
冷水冲面 → 轻拍干 → 须后产品 → 保湿（可选）
   ①         ②         ③          ④
```

## 第一步：冷水冲面

剃须完成后，立即用 **冷水** 冲洗面部：

- 冷水 **收缩毛孔**，减少感染风险
- 冲掉残留的泡沫和断须
- 持续冲洗 **15-20 秒**

## 第二步：轻拍干

- 用干净毛巾 **轻拍** 吸干水分，**不要擦拭**
- 擦拭动作会摩擦刚剃完的敏感皮肤
- 拍干后皮肤微湿时最适合涂抹须后产品

## 第三步：须后产品选择

### 须后水（Aftershave Splash）

| 特性 | 说明 |
|------|------|
| 质地 | 液体，含酒精 |
| 作用 | 消毒杀菌、收敛毛孔、清爽感 |
| 适合 | 油性皮肤、夏季、喜欢清爽感 |
| 注意 | 酒精刺痛属正常，但过度刺痛说明剃须手法过重 |

### 须后膏（Aftershave Balm）

| 特性 | 说明 |
|------|------|
| 质地 | 乳液/膏状 |
| 作用 | 保湿修复、舒缓刺激、滋养皮肤 |
| 适合 | 干性/敏感皮肤、冬季、剃须刺激较大时 |
| 注意 | 用量不宜过大，避免油腻感 |

### 其他产品

| 产品 | 用途 | 用法 |
|------|------|------|
| **明矾块（Alum Block）** | 止血、消炎、反馈剃须质量 | 打湿后涂抹，30秒后冲洗 |
| **须后凝胶** | 冰爽镇静 | 冷藏后使用效果更佳 |
| **单涂保湿霜** | 极简护理 | 适用不常剃须的皮肤 |

## 第四步：保湿（可选）

如果须后产品本身不含保湿成分（如纯须后水），建议后续加涂保湿霜：

- 无香精保湿霜为佳
- 薄涂一层即可
- 锁住皮肤水分

## 剃须质量反馈

明矾块是检验剃须质量的工具：

| 明矾反馈 | 含义 |
|-----------|------|
| 轻微刺痛 | 正常，该区域有微小刮擦 |
| 明显刺痛 | 该区域剃须压力过大或角度不对 |
| 刺痛+红点 | 有割伤，下次需要更注意手法 |
| 几乎无感 | 剃须手法理想 |

## 刺激缓解

| 症状 | 处理方法 |
|------|----------|
| **剃须灼热感** | 冷敷 5 分钟 + 芦荟胶 |
| **剃须红疹** | 停剃 2-3 天 + 温和保湿 + 须后膏 |
| **内生毛（Ingrown Hair）** | 温和去角质 + 停剃该部位至皮肤恢复 |
| **割伤** | 明矾块按压止血 / 止血笔 / 冷水 |

## 常见错误

| 错误 | 后果 |
|------|------|
| 热水冲面 | 刺激刚剃完的皮肤 |
| 用力擦拭 | 摩擦引起红疹 |
| 涂抹含酒精产品后暴晒 | 光敏感 + 色素沉着 |
| 不涂任何产品 | 皮肤干燥、起皮 |
| 剃须后立即去角质 | 严重刺激 |
