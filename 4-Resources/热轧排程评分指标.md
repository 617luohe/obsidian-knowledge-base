# 热轧排程评分指标

> 排程质量评估的 23 项指标体系，涵盖物料类型、轧宽、轧厚、出钢标记、冷热坯、公里数六个维度。由 `PlanScorer` 在排程完成后自动计算。

关联项目：[[冷热集批排产项目]]

## 评分项总览

### 物料类型

| 指标 | 计分 | 方向 |
|------|------|------|
| 主轧数 | +1/卷 | 鼓励多排主轧材 |
| 可利用材数 | -1/卷 | 惩罚过度依赖可利用材 |
| 主轧段过渡材数 | -0.5/卷 | 减少主轧段内不必要的过渡 |

### 轧宽

| 指标 | 计分 | 说明 |
|------|------|------|
| 主轧段轧宽违规 | -5/处 | B 类宽度违例 |
| 主轧段轧宽趋势 | -2/离散点 | 偏离线性趋势的离散点 |
| 主轧段轧宽反跳 | -1/处 | 宽度反向跳变 |
| 过渡段轧宽趋势 | -2/离散点 | 过渡段宽度趋势离散度 |
| 过渡段轧宽反跳 | -1/处 | 过渡段宽反向跳变 |
| 轧宽尖点 | -2/处 | 当前物料宽于前后两卷 |
| 轧宽同规则集中 | -1/段 | 相同宽度连续集中 |

### 轧厚

| 指标 | 计分 | 说明 |
|------|------|------|
| 主轧段轧厚违规 | -5/处 | B 类厚度违例 |
| 主轧段轧厚趋势 | -2/离散点 | 偏离线性趋势的离散点 |
| 主轧段轧厚反跳 | -1/处 | 厚度反向跳变 |
| 过渡段轧厚趋势 | -2/离散点 | 过渡段厚度趋势离散度 |
| 过渡段轧厚反跳 | -1/处 | 过渡段厚反向跳变 |
| 轧厚尖点 | -2/处 | 当前物料厚于前后两卷 |
| 轧厚同规则集中 | **+0.5/卷** | 鼓励同厚度集中 |

### 出钢标记

| 指标 | 计分 | 说明 |
|------|------|------|
| 出钢标记违规 | -5/处 | B 类出钢标记违例 |
| 出钢标记集中 | **+0.5/卷** | 鼓励同标记集中 |

### 冷热坯

| 指标 | 计分 | 说明 |
|------|------|------|
| 热装数量 | **+2/卷** | 鼓励热装（H），权重最高 |
| 冷热交替 | -1/处 | 减少 C/H 切换 |

### 公里数

| 指标 | 计分 | 说明 |
|------|------|------|
| 辊期公里数合规 | -1/超过 | 超出 C 类规则约束 |
| 烫辊过渡段合规 | -0.5/超过 | 超出 C 类规则约束 |

## 设计原则

1. **正向激励**：主轧材、同厚度集中、同出钢标记集中、热装 —— 加分为正
2. **负向惩罚**：违规、反跳、尖点、冷热交替 —— 扣分制
3. **趋势优先**：通过线性拟合评估宽/厚趋势，鼓励平滑过渡
4. **热装优先**：热装（+2/卷）是单项权重最高的指标，反映节能降本导向
