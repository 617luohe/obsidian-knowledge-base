# Vegas 交易系统（罗晟）

> 基于 EMA 均线组的趋势跟踪交易系统，以 144/169 EMA 为核心构建通道，结合 12/24 EMA 触发线和多周期分析进行交易决策。

## 系统概述

Vegas 系统是完全基于指数移动平均线（EMA）的趋势跟踪系统，由罗晟在《Vegas 交易系统》中系统化阐述。其核心思想是：

- 用 **144/169 EMA** 构成"Vegas 通道"判断趋势方向
- 用 **12/24 EMA** 作为入场触发信号
- 通过 **多周期共振** 确认交易信号的有效性
- 以 **通道边界** 作为动态支撑/阻力位管理风险

## 核心组件

| 组件 | EMA 参数 | 作用 |
|------|----------|------|
| Vegas 通道 | 144 + 169 | 趋势方向判断、动态支撑/阻力 |
| 趋势生命线 | 144 | 主要趋势线，价格在线上看多、线下看空 |
| 确认线 | 169 | 与144构成通道宽度，确认趋势强度 |
| 触发线 | 12/24 | 短线入场时机判断，突破/回踩信号 |
