# 2026年上半年工作总结 — 陈昱慧

> 原始文档，已整理为 [[2026年上半年工作总结.pptx|PPT汇报]]。

## 工作概述

### 1. 天车定位辅助系统项目

**接口设计与开发（4月）**
- 讨论定义地面系统前后端接口形式，完成报表、库存、规则配置等页面API接口定义及文档编写。
- 开发报表页面、规则配置页面、库管页面后端API接口。
- 开发地面系统天车坐标实时接收及转发功能、天车信息显示接口。

**终端功能开发与联调（4月）**
- 开发指令执行重量判断、位置判断逻辑，开发指令锁定解锁功能。
- 修改终端停车位布局及界面风格，按详细设计书1:1还原。
- 开发配料计划分解功能：根据天车作业范围分解配料计划，终端只显示作业范围内指令，点击后每种物料生成一条指令。
- 与前端联合测试修改指令状态、配斗准确率报表查询、查询计划接口等功能。

**MQ通讯与串口开发（5月）**
- 开发料场、物流与天车MQ报文通讯与解析功能。
- 开发天车终端重量实时显示UDP报文解析功能及串口程序转发实时重量接口。
- 测试天车终端坐标串口解析，优化坐标刷新及报文粘包解析。
- 配合现场测试天车秤重量串口、定位与称的串口解析及终端界面显示。
- 配合现场测试天车坐标采集接口程序，调整定位系统基站解析逻辑，只解析指定标签数据。

**智慧物流与PES接口开发（5月）**
- 开发车辆匹配停车位、发送停车位基础信息、到位开始作业信息等接口。
- 开发废钢存放区域修改信息、天车重量信息上报、停车位装车完成推送物流MQ接口。
- 开发完善PES系统废钢计划下发接口。

**界面优化与数据库对齐（5月）**
- 修改终端界面布局，优化重量显示、指令完成弹窗及锁定解锁逻辑。
- 完成6项UI/逻辑优化：箭头加宽、称重大字、同车单锁、天车号列、停车位PT编码映射等。
- 按新版Oracle表结构对齐，修改字段映射、查询关联及建表脚本。
- 修改完善智慧物流MQ接口解析程序。

**集中测试与功能完善（6月）**
- 与物流、料场联合测试MQ报文收发及与天车终端联动测试。
- 修复终端计划待装量计算错误、停车位编号显示、选中取消等问题。
- 完善库位校验偏移量逻辑，修改计划结束逻辑（点击结束+重量验证）。
- 配合前端测试地面系统，部署程序至测试服务器。

### 2. 质检部实验室项目

**数据库搭建与接口开发（4月）**
- 安装Oracle数据库，梳理表结构完成本地建表。
- 开发人员档案页面功能，包括附件上传模块、审批模块，完成明细表及审批表建表。
- 人员监督页面前后端联调，测试完善页面功能及操作历史写入。
- 开发人员、设备、方法基础表单的增删改查接口API。
- 与业主讨论功能修改需求，调整表结构并编写Excel模板。

**人员监督模块（6月）**
- 开发人员监督页面后台逻辑（页面查询、详细信息查询、功能修改、附件上传下载等）。
- 与前端联调测试人员监督页面，修复时间戳筛选功能。
- 与业主开会讨论功能修改需求，讨论人员监督页面功能与设计。

### 3. 冷轧排程项目

- 辅助排查导入计划Excel表字段为空时报错问题（4月）。
- 开发整点定时同步库存功能并转发给算法模型，与算法联调测试（6月）。
- 排查修复自动化率计算的主轧材统计问题（6月）。

### 4. 4300水处理项目

- 设计前端展示页面初版（4月）。

### 5. 其他工作

- 低代码平台技术文档学习、环境安装、设计器使用培训及页面功能练习。
- 参加智工工业大模型专题交流。
- 参加研究所《基于Langchain框架的智能体开发实践分享》学习交流与实践。
